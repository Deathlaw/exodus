Exodus branding instructions:

  1. The easiest way to do these is to check out from cvs.
  2. Get or build a copy of these files:
    • Exodus.exe
    • IdleHooks.dll
    • ZipDll.dll
    • exodus.nsi
    • NSIS 2
  3. These files you might want to host on your own web server:
    • indy_openssl096g.zip
    • richupd.exe
  4. Create a branding.xml file.  Here is a template, with the defaults.  Any preference you set in this file will become the default.  You can set other preferences in here as well, such as these, or things that you find in your exodus.xml file.  Good candidates are fonts and colors.
  5. Move ZipDll.dll to $NSIS_HOME\Plugins.
  6. Read the NSIS docs, to find out how to customize exodus.nsi.  I'd suggest removing the "Bleeding-edge updates" section, and adding branding.xml to the !Exodus section.
  7. If you want your branding file to auto-update, put the file on a web server, and set the <branding_url/> option.
  8. Build your setup.exe file.  I use: "makensis.exe /v2 exodus.nsi"
  9. Distribute your new setup.exe.
